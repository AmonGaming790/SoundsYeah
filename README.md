#

🔊 SoundsYeah

**SoundsYeah** is a stylish and lightweight Python application inspired by the Minecraft mod *CreamyKeys*. Unlike the original mod, this application works **system-wide**, bringing satisfying mechanical sounds to your entire OS experience.

---

## ✨ Features

* **Custom Sound Packs:** Independently select different sound profiles for your mouse and keyboard.
* **Smart Settings:** Remembers your volume and selected packs between sessions.
* **Modern GUI:** Clean Dark Mode interface built with CustomTkinter.
* **System Tray Integration:** Runs quietly in the background without cluttering your taskbar.
* **Animated Interface:** Features an interactive close button and responsive design.
